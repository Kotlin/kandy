inline fun LayerCollectorContextMutable.bars(block: BarContextMutable.() -> Unit)

Adds a new bars layer.

Creates a context in which you can create bindings using aesthetic attribute properties invocation. In this context, you can use mutable mappings - that is, do mapping and scaling with iterables.

Aesthetic attributes:

Positional:

  • x

  • y

Non-positional:

  • color - bars fill color, of the type Color, mappable. (TODO grouping)

  • alpha - layer alpha, of the type Double, mappable. (TODO grouping)

  • width - bars width, of the type Double, mappable without an explicit scale. (TODO grouping)

  • borderLine.type - borderline type, of the type LineType, mappable. (TODO grouping)

  • borderLine.width - borderline width, of the type Double, mappable. (TODO grouping)

  • borderLine.color - borderline width, of the type Double, mappable. (TODO grouping)

bar {
   x(listOf(1.0, 2.2, 3.1, 4.0, 4.9)) // mapping from `time` column to `X` with the default scale

   borderLine.width(2.5) // // setting of constant `borderLine.width` value
}
