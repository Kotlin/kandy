Layer

data class Layer(val dataset: TableData?, val geom: Geom, val mappings: Map<AesName, Mapping>, val settings: Map<AesName, Setting>, val features: Map<FeatureName, LayerFeature> = emptyMap(), val freeScales: Map<AesName, FreeScale> = emptyMap())

Layer is a collection of data and mappings from it. It is characterized by its Geom.

Parameters

dataset

the dataset of this layer. In case it is null, the data for this layer is taken from the dataset of the Plot

geom

the Geom that describes this layer.

mappings

the Map of the mappings; keys are aesthetic attributes, values are mappings on corresponding attributes.

settings

the Map of the setting; keys are aesthetic attributes, values are setting of corresponding attributes.

features

the Map of the layer features; keys are feature names, values are features with corresponding names.
