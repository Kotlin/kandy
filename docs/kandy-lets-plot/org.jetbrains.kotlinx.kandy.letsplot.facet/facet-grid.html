facetGrid

fun LayerPlotContext.facetGrid(x: ColumnReference<*>, y: ColumnReference<*>, scalesSharing: ScalesSharing? = null, xOrder: OrderDirection = OrderDirection.ASCENDING, yOrder: OrderDirection = OrderDirection.ASCENDING, xFormat: String? = null, yFormat: String? = null)

Splits data by two faceting variables across X and Y. For each data subset creates a plot panel and lays out panels as grid. The grid columns are defined by X faceting variable and rows are defined by Y faceting variable.

Parameters

x

variable which defines columns of the facet grid.

y

variable which defines rows of the facet grid.

scalesSharing

specifies whether scales are shared across all facets.

xOrder

specifies ordering direction of columns

yOrder

specifies ordering direction of rows

xFormat

specifies the format pattern for displaying faceting values in columns.

yFormat

specifies the format pattern for displaying faceting values in rows.

Format pattern in the xFormat/yFormat parameters can be just a number format (like "d") or a string template where number format is surrounded by curly braces: "{d} cylinders". Note: the "$" must be escaped as "\$"

Examples: ".2f" -> "12.45" "Score: {.2f}" -> "Score: 12.45" "'Score: {}' "-> "Score: 12.454789"
