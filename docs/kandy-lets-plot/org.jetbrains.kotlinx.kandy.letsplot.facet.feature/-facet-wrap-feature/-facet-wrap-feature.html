FacetWrapFeature

constructor(facets: List<String>, nCol: Int?, nRow: Int?, orders: List<OrderDirection>, scalesSharing: ScalesSharing, direction: Direction, formats: List<String?>)

Parameters

facets

list of faceting variable names.

nCol

number of columns.

nRow

number of rows.

scalesSharing

specifies whether scales are shared across all facets.

orders

list of OrderDirection values specifying ordering direction panels. The order values are positionally matched to variables in facets.

formats

list of strings specifying the format pattern for displaying faceting values. The format values are positionally matched to variables in facets.

direction

direction of the facet.

Format patterns in the format parameter can be just a number format (like "d") or a string template where number format is surrounded by curly braces: "{d} cylinders". Note: the "$" must be escaped as "\$" For more info see: https://github.com/JetBrains/lets-plot-kotlin/blob/master/docs/formats.md

Examples: ".2f" -> "12.45" "Score: {.2f}" -> "Score: 12.45" "Score: {}" -> "Score: 12.454789"
