facetGridY

fun LayerPlotContext.facetGridY(y: ColumnReference<*>, scalesSharing: ScalesSharing? = null, order: OrderDirection = OrderDirection.ASCENDING, format: String? = null)

Splits data by a variable across Y. For each data subset creates a plot panel and lays out panels as grid.

Parameters

y

Variable which defines rows of the facet grid.

scalesSharing

Specifies whether scales are shared across all facets.

order

Specifies ordering direction of rows

format

Specifies the format pattern for displaying faceting values in rows.

Format pattern in the format parameters can be just a number format (like "d") or a string template where number format is surrounded by curly braces: "{d} cylinders". Note: the "$" must be escaped as "\$"

Examples: ".2f" -> "12.45" "Score: {.2f}" -> "Score: 12.45" "'Score: {}' "-> "Score: 12.454789"
