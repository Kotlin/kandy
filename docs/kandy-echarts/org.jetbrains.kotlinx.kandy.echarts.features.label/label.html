fun LineContext.label(block: LabelContext.() -> Unit)

Label options settings for line. If a property isn't set or set to null, a default value will be used.

  • position - label position. Includes position, distance and rotation settings. LabelPosition.position may not apply to some types of plots. By default, position is top, distance is 5, rotate isn't defined

  • formatter - data label formatter, supports string template.

String template

Model variation includes:

  • {a} - layers name.

  • {b} - the name of a data item.

  • {c} - the value of a data item.

  • {@xxx} - the value of a column named xxx, for example, {@product} refers the value of product column.

  • {@[n]} - the value of a column at the index of n, for example, {@[3]} refers the value at column3.

  • textStyle - text style settings

  • border - border settings

plot {
 line {
     label {
         position = LabelPosition.top(distance = 5, rotate = 45)
         formatter = "{b}: {@score}"
         textStyle.color(Color.BLUE)
         border {
             color = Color.RED
             width = 1.5
         }
     }
 }
}
